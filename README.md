- 前例がないことを切り開くこと
- ビジネス・エンジニアリング・チームのそれぞれ多角的な視座で考えて判断して行動すること
- 新規事業の立ち上げ
- クリティカルシンキング
- よいチームを作ること
- チームや組織の文化をつくること
- 人を巻き込むこと
- 人を元気にすること／自分が元気であること
- エンジニアコミュニティに発信すること
